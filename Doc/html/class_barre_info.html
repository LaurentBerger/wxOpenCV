Detailed Description

classe permettant d'accéder à des informations sur l'image

La classe BarreInfo permet d'afficher la valeur du pixel où est le curseur, d'afficher un zoom sur l'image, activer des statistiques sur l'image, de remplacer l'image par l'entrée d'une caméra.
