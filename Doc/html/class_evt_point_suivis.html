Detailed Description

la classe définit un les caractéristiques d'un évènement

La classe EvtPointSuivis définit un nouveau type d'évènement utilisé pour informer la fenêtre qu'une nouvelle image issue d'un flux vidéo est disponible. L'évènement est envoyé à la fenêtre gérant l'affichage du flux vidéo.
