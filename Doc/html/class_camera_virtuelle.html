Detailed Description

classe permettant de capturer un flux vidéo

La classe CameraVirtuelle permet de capturer un flux vidéo reconnu par OpenCV ou par une caméra Andor(non maintenu). Un thread est crée envoyant un évenement à la fenêtre affichant l'image pour indiquer qu'une nouvelle image est disponible.
